This is a list of all the constructors for the vanilla monsters. The parameters are what you pass to the "Parameters" field in your content.json file. e.g.: 

{
  "Format": "2.9.0",
  "Changes": [
    {
      "Action": "EditData",
      "Target": "aedenthorn.CustomMonsters/dict",
      "Entries": {
      "MyCustomMonster": {
        "Type": "Serpent",
        "Parameters":"position,name",
        "Name":"Royal Serpent"
      }
    }
  ]
}

For switch, I have included in parenthesis the name of the switch that will toggle if you set "Switch": true for that monster type.

See doc.md for more info.
