This is a list of all the constructors for the vanilla monsters. The parameters are what you pass to the "Parameters" field in your content.json file to tell the mod which C# constructor to use. e.g.: 

{
  "Format": "2.9.0",
  "Changes": [
    {
      "Action": "EditData",
      "Target": "aedenthorn.CustomMonsters/dict",
      "Entries": {
      "MyCustomMonster": {
        "Type": "Serpent",
        "Parameters":"position,name",
        "Name":"Royal Serpent"
      }
    }
  ]
}

This will call the constructor for Serpent that has two parameters (position and name). The position parameter will be passed by whatever mod is spawning the monster. Here I supply the name parameter with "Name": "Royal Serpent", which tells the game to make a royal serpent type monster.

For "name" I have included in parentheses names used by the game to create specific monster types. Unless you are intent on basing your monster off of one of these special types, you probably shouldn't use constructors that have the "name" parameter.

For switch, I have included in parenthesis the name of the switch that will toggle if you set "Switch": true for that monster type.

See doc.md for more info.
